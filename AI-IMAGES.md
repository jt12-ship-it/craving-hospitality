# Craving Hospitality — AI Image Prompt Pack

This site is built so **every photo is a fixed filename in `assets/img/`**. Generate a crisp image,
save it with the **exact filename** below, drop it into `assets/img/`, and the site upgrades instantly —
no code changes. Square (1:1) or 4:3 works for all; the CSS crops to fit.

> Works with ChatGPT / DALL·E 3, Midjourney, Google Gemini/Imagen, Flux, Ideogram, etc.
> For Midjourney, append `--ar 1:1 --style raw --v 6` (sandwiches: `--ar 4:3`).
